# HAZARD OS | INTEGRATED PORTFOLIO SYSTEM
### SYSTEM OVERVIEW
**Hazard OS** is a high-tech terminal interface built with Streamlit. It serves as the primary deployment environment for(Integrated Barangay Management System) and showcases technical milestones in software automation and community data management.
